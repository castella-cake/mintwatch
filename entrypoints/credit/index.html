<MintWatch について

About

現在使用中のバージョン:
MintWatch は、ニコニコ動画用の視聴することにフォーカスした代替フロントエンドです。
MintWatch は MIT License のもとで利用することができます。

Website/X では MintWatch の更新情報も投稿しています。
(Website の投稿通知は、ActivityPub で受け取ることもできます)

Created by CYakigasi
Website Niconico SNS Links(Misskey,Bluesky,Twitter...)

Discord

PepperMint+ / MintWatch の Discord サーバーがあります。
バグ報告やフィードバックをより簡単に行ったり、ベータ版や今後の開発に関するアンケートなどに参加できます。
ニコニコ関連の UserStyle や UserScript などに関するディスカッションも歓迎します！
NicoPM Community Discord に参加する

Contribute

MintWatch の Github リポジトリがあります。
機能追加やバグ修正などのPull requestは大歓迎です。
バグを見つけた場合はissueに報告していただけると助かります。
https://github.com/castella-cake/mintwatch

Donate

Github Sponsors から開発者に$1~/月もしくは$2~から寄付できます。
MintWatch を気に入っていただけたなら、よければ支援をお願いします。
https://github.com/sponsors/castella-cake

Special Thanks

MintWatch のコメントレンダーは niconicomments を使用しています。
その他にも、hls.js や React などのライブラリを使用しています。
これらのライブラリの助けがなければ、MintWatch は成立しませんでした。改めてライブラリの開発者に感謝します。
そして、MintWatch の使用者や貢献者、寄付者に感謝します。
