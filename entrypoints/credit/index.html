<MintWatch について

About

現在使用中のバージョン:
MintWatch は、ニコニコ動画の視聴そのものにフォーカスした、カスタマイズ性の高い動画プレイヤーです。
MintWatch は MIT License のもとで利用することができます。

Website/X(旧Twitter)では MintWatch の更新情報も投稿しています。
(Websiteの投稿通知は、ActivityPubで受け取ることもできます)

記事掲載は大歓迎ですが、事後報告でも大丈夫なのでWebsiteの問い合わせまたはDMで一報くださると幸いです
Created by CYakigasi Website Niconico Discord SNS Links(Misskey,Bluesky,Twitter...)

Contribute

MintWatch のGithubリポジトリがあります。

Special Thanks

MintWatch のコメントレンダーは niconicomments を使用しています。
その他にも、hls.js や React などのライブラリを使用しています。
これらのライブラリの助けがなければ、MintWatch は成立しませんでした。改めてライブラリの開発者に感謝します。
そして、MintWatch の使用者や貢献者、寄付者に感謝します。
